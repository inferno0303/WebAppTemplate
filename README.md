# 介绍

该项目是一个前后端分离项目（WebApp）的工程模板，包含登录、注册、账户管理的基本代码，基于 SpringBoot 和 Vue.js 框架，自用😄

```
version: v20230814
author: inferno0303@github
```

## 如何使用

### 前端

`frontend`文件夹就是前端的静态目录

### 后端

`backend`文件夹就是SpringBoot工程目录


### 数据库
`sqlite3.db`文件夹下就是数据库目录
